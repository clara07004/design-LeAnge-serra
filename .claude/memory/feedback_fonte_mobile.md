Escala de fonte aprovada para slides Instagram (canvas 1080×1350, leitura mobile).

**Why:** texto pequeno demais é ilegível no feed; fonte fora da escala quebra o reconhecimento
de marca entre posts.

**How to apply:** usar a tabela. **Se o texto não couber, corte texto — nunca reduza a fonte
abaixo do padrão.**

| Elemento | Tamanho | Peso |
|---|---|---|
| Punchline / display | 92–96px | 700 |
| Headline slide | 84–90px | 700 |
| Body principal | 44–46px | 400 |
| Body secundário | 40–42px | 300 |
| Subtítulo / nota | 40px | 400 |
| CTA button | 24px | 700 |
| Caption | 28px | 400 |
| Label uppercase | 18px | 700 (Montserrat) |
| Tagline header | 16px | 700 (Montserrat) |

**Famílias:** conforme `marca/DESIGN.md` (`typography.display.fontFamily` + `typography.body.fontFamily`).
Máximo 2 famílias. Sem serifa em digital. Headline de referência: display 88px / 700 /
line-height 1.06 / letter-spacing -0.022em. Ver [[feedback-carousel-design-aprovado]].
